## 特性
* 浏览 `Edge` 、 `Edge Update` 、 `Edgev Webview2` 的所有可用策略。
* 配置类型为 `String` 、 `Boolean` 、 `Integer` 的策略。
* 多语言支持。可用的显示语言有 `en-US` 、 `zh-CN` 、 `zh-TW` 。

## 说明
* 默认值根据文档内容推断，仅供参考。
* 暂不支持主动切换显示语言。
